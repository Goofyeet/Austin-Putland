    Calculator Project

    Description: In this project I implemented a new engineering notation feature. This allows a user to make calculations that are less error prone by displaying them in their ideal format. Engineering notation means that a number will be formatted in scientific notation, however the power will always be divisible by 3. This can be more convenient for the user if SI prefixes (micro, milli, kilo, giga, etc.) are used very often in their calculations.

    IR Remote Project

    Description: I wanted a second remote for my fan, so I decided to clone the original remote. With the help of an Arduino and an open source IR library I was able to figure out the encoding method used, pulse distance encoding. After that, I used an oscilloscope to figure out the codes being sent for each button. Once the codes were determined, I wrote AVR assembly code for an ATTiny 12V microcontroller. The assembly code and project files can be found on my Github repo using the link below.

    Snake Game

    Description: I wanted to make a small game to learn C++ a bit more. I decided to make a snake game in the console, but after making it I wanted it to look better. So I thought the next best thing would be to learn how to create a GUI. I decided to create it with the SFML library and very simple textures I created in Gimp. Additionally, I wanted to learn unit testing so I used the open source Catch2 library. Also, I then wanted to understand build systems so I taught myself how to integrate CMake into my project. I then realized I was succumbing to feature creep...
